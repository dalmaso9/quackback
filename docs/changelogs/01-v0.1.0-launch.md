# v0.1.0 - Launch

# Published: 2026-02-12

## Title

Featurepool is live - open-source customer feedback platform

## Content

We're excited to announce the first public release of Featurepool, the open-source alternative to Canny, UserVoice, and Productboard.

What's included:

Feedback boards - Public voting, status tracking, comments, reactions, and duplicate detection. Create multiple boards to organize feedback by product area.

Admin inbox - Unified triage view with filtering by status, board, tags, votes, and response state. Bulk actions and analytics to stay on top of incoming feedback.

Roadmap and changelog - Show your users what's planned and close the loop when features ship. Changelog entries can be linked to shipped feedback posts.

Embeddable widget - Collect feedback from inside your app with a single script tag. Configurable placement, board targeting, and theme matching.

14 integrations - Slack, Linear, Jira, GitHub, Asana, ClickUp, Shortcut, Azure DevOps, Intercom, Zendesk, HubSpot, Discord, Teams, and Zapier. Two-way sync keeps statuses in sync with your issue tracker via inbound webhooks.

REST API and MCP server - API keys, outbound webhooks, and a Model Context Protocol server for AI agents. 7 MCP tools out of the box.

Authentication - Email OTP and configurable OAuth providers (Google, GitHub, and more).

Self-host anywhere - Docker, Railway one-click deploy, S3-compatible storage, and any SMTP provider. Your data stays on your infrastructure.
