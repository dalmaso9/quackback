# v0.5.1 - Interactive developer guides

# Published: 2026-03-11

## Title

Interactive developer guides and settings redesign

## Content

All developer-facing settings pages have been redesigned with interactive guides and live code examples.

API Keys quick start - Multi-language code examples (curl, JavaScript, Python, Go, PHP) with syntax highlighting, base URL copy, and API resources summary.

Webhooks verification guide - HMAC-SHA256 signature verification examples in 5 frameworks (Node.js, Python, Ruby, Go, PHP), webhook headers reference, and event badges.

MCP setup guide - Interactive client selector with icons for Claude Code, Cursor, VS Code, Windsurf, and Claude Desktop. Dark code panels with OAuth/API key variant toggle.

Widget settings overhaul - Two-column layout with live preview panel, dark syntax-highlighted code blocks, and copy-to-clipboard embed snippets.

Vote management - Admins can remove any vote (direct or proxy) from the voters modal. New proxy_vote MCP tool and DELETE endpoint for the REST API.

Widget simplification - Icon-only trigger button and simplified panel layout with close button on the home screen.
