# v0.4.0 - AI-powered duplicate detection

# Published: 2026-03-01

## Title

AI-powered duplicate detection and feedback aggregation

## Content

Featurepool now automatically finds duplicate posts using a hybrid AI approach - vector similarity search combined with LLM verification.

AI duplicate detection - When new feedback arrives, Featurepool searches for similar posts using embeddings and full-text search, then verifies with an LLM. Merge suggestions appear in the admin inbox with accept/dismiss actions. A periodic sweep catches duplicates that arrive close together.

AI post summaries - Each post gets an automatically generated summary with key quotes, suggested next steps, and live staleness detection. Summaries refresh when significant new comments arrive.

Feedback insights tab - A dedicated insights view in the admin feedback section shows AI-generated suggestions including duplicates and new feedback patterns, with filtering by type and sorting by relevance.

Comment count filter - New minimum comments filter in the admin inbox, matching the existing minimum votes filter.

Admin UI refresh - Unified visual patterns across the inbox, insights, changelog, and users pages.
