# v0.3.0-v0.3.11 - SEO, delete/restore, and invitations

# Published: 2026-02-27

## Title

SEO, post delete/restore, and invitation flow rewrite

## Content

A series of improvements to SEO, content management, and the team invitation experience.

SEO meta tags and sitemap - All portal pages now render Open Graph and Twitter Card meta tags with canonical URLs. A dynamic sitemap.xml includes all public posts and changelog entries, and robots.txt blocks admin, auth, and API paths.

Post delete and restore - Admins can soft-delete posts from the post modal and restore them within a 30-day window. A "Deleted posts" filter in the inbox shows recently deleted posts. MCP gets matching delete_post and restore_post tools.

Invitation flow rewrite - Redesigned the accept-invitation page with a welcome screen, atomic claim to prevent race conditions, email ownership verification, and server-side password setup.

Date filtering - dateFrom and dateTo parameters on the MCP search tool and public API enable date-range queries.

Custom OIDC OAuth providers - Support for generic OAuth2/OIDC providers beyond the built-in social logins. Configure custom providers with a discovery URL, client ID/secret, and display name.

Bug fixes:

- Fixed soft-deleted posts and comments appearing on the public portal
- Fixed the Responded/Unresponded inbox filter crash
- Fixed widget and RSS feed generating http:// URLs behind Cloudflare TLS termination
- Fixed settings page URLs in reverse proxy deployments
- Fixed OAuth email scope for ChatGPT and other OIDC-aware MCP clients
- Fixed post descriptions not displaying for plain-text posts in the admin modal
