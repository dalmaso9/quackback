# v0.1.1-v0.1.6 - Self-hosting improvements

# Published: 2026-02-13

## Title

Self-hosting improvements and deployment fixes

## Content

A round of fixes to make self-hosting smoother, especially for Docker and Railway deployments.

Optional email setup - Featurepool now gracefully degrades when no email provider (SMTP or Resend) is configured. The Email OTP sign-in option is automatically hidden, and invitation functions return the invite link directly so admins can share it manually.

S3 proxy mode - New S3_PROXY environment variable that streams file bytes through the server instead of issuing a redirect to a presigned URL. Fixes broken avatars and images when the browser can't reach the S3 endpoint directly, common with ngrok tunnels, HTTPS apps with HTTP MinIO, or Docker networking.

Resilient config validation - Empty environment variable strings (common in Docker Compose) no longer cause config validation failures for optional fields.

Bug fixes:

- Fixed a deployment-breaking bug where the telemetry_config migration was never applied, causing 500 errors on every request
- Fixed image serving for private S3 buckets (e.g. Railway Buckets) by defaulting to the presigned URL proxy
- Fixed Playwright e2e specs being picked up by vitest
