# v0.5.0 - Slack integration and AI signals

# Published: 2026-03-10

## Title

Slack integration, AI signals, and public API expansion

## Content

A major release bringing Slack as a feedback source, contextual AI insights, and significant API expansion.

Slack integration - Two new ways to capture feedback from Slack. The "Send to Featurepool" message shortcut lets team members forward any message as feedback with one click. Channel monitoring auto-ingests messages from configured channels into the feedback pipeline.

AI signals - AI-generated insights are now surfaced contextually in the post modal via signal badges, a summary bar, and a similar posts card. Duplicate signals appear inline with merge/dismiss actions, eliminating the need to navigate to the Insights tab.

Bidirectional merge from post modal - A new Merge dropdown offers "Merge others into this" (multi-select with AI suggestions) and "Merge into existing" directions. An amber dot badge appears when pending AI duplicate signals exist.

Activity timeline - Every post now has an Activity tab showing status changes, merges, tag changes, owner assignments, proxy votes, and other events in reverse chronological order.

Proxy voting - Admins can vote on behalf of users with full attribution tracking. The voters modal shows who added each vote.

Author selector - Admins can choose the post author when creating posts, with a searchable member dropdown and inline user creation.

Public API expansion - Five new REST endpoints for managing AI suggestions and viewing post activity. MCP server grows from 17 to 22 tools. Existing endpoints now return summary, merge, and lock fields.

Feedback tab consolidation - The standalone Incoming page is now a sub-tab under Feedback, reducing sidebar nav items and keeping all feedback workflows in one place.
