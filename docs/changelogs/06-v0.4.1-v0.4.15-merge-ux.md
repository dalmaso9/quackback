# v0.4.1-v0.4.15 - Merge UX, branding, and insights

# Published: 2026-03-04

## Title

Merge UX overhaul, gold branding, and scanner-safe invitations

## Content

Major UX improvements to duplicate merging, new Featurepool brand identity, and fixes for corporate email environments.

Redesigned merge flow - Merge suggestions now show stacked cards with a merged-result preview, a confirmation dialog, and a post-merge resolved state with inline undo. PMs can click the preview card for a full readonly preview including comments from both posts.

Bidirectional merge - Swap which post gets merged into which with a single toggle.

Gold brand identity - Default primary color updated from indigo to gold to match the Featurepool website brand. Applied across the app theme, email templates, and invite pages.

Scanner-safe magic links - Corporate email scanners (Microsoft Safe Links, Proofpoint, Mimecast) can no longer invalidate invitation magic links. An intermediate branded landing page sits between the email link and the auth endpoint.

Link-based password reset - Replaced OTP-based forgot-password with email link-based reset using 24-hour token expiry.

Profile password management - Users can change their password or set one for the first time from profile settings.

Voter subscription controls - Admins can view and change notification subscription levels for individual voters directly from the voters modal.

Structured logging - Added consistent log prefixes across all domain services and server functions for better observability.

Upgraded AI models - All LLM services now use Gemini 3.1 Flash for improved quality across extraction, summaries, and merge suggestions.
